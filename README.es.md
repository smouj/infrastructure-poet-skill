# Poeta de Infraestructura

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad Poeta de Infraestructura para OpenClaw.

## Qué Soluciona
Transforma la infraestructura como código en planos intuitivos y legibles.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales listos para producción
- Enfoque primero en seguridad
- Soporte de reversión

## Uso
/infrastructure-poet

## Ejemplo
```
Analizar código para problemas de devops.
```

Esta habilidad está disponible automáticamente en OpenClaw.
